Vous venez de décompresser l'archive InclusionDetector?


Pour installer le système de détection d'inclusion, il est nécessaire de suivre
la procédure suivante:

- Installer Python2.7.13.msi
- Installer numpy-1.10.1-win32-superpack-python27.exe
- Installer matplotlib-1.3.0.win32-py2.7.exe
- Installer opencv-3.2.0-vc14.exe

Ca y est l'ensemble des éléments nécessaires pour l'utilisation de l'outil
"Inclusion Detector"
